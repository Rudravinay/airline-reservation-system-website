About This Project

This website is a front-end simulation of an Airline Reservation System originally implemented in C++ using Object-Oriented Programming.

  • 30 seats (L1–L15 and R1–R15)
  • Classes: First Class (1–5), Business (6–10), Economy (11–15)
  • Book / cancel seats and view passenger list

The original C++ program uses: classes, encapsulation and abstraction to manage the reservation system in a console application.
