About This Project

This web UI is a modern front-end for an Airline Reservation System originally implemented in C++ using Object-Oriented Programming concepts.

The original console application uses classes like Seat and AirlineReservationSystem to manage bookings, cancellations, and passenger details.

Key Features

  • 30 seats: L1–L15 and R1–R15
  • Seat categories: First, Business, Economy
  • Interactive booking and cancellation
  • Live passenger list table

OOPS Concepts Used

  • Classes & Objects
  • Encapsulation (private data)
  • Abstraction of booking logic

This UI is built with pure HTML, CSS, and JavaScript and hosted on GitHub Pages.
